## 简单的资源共享

下面就介绍Python当中一个比较实用的功能，不需要使用到其他的第三方库，**只需要一条命令就可以实现局域网内的计算机资源共享，类似于Windows上的共享文件夹。**

```
# 打开命令行，切换到共享的文件夹
文件夹所在盘符:\
cd 文件夹所在盘符:\绝对路径

# 输入下面命令，启动一个简单的http服务器
python -m http.server
```
